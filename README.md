# InstaClone Backend
Instagram Clone example. The script working with:
- NodeJS
- Express
- MongoDB ( Remote MongoDB Atlas )
- Socket.IO

## Requiriements to work
- NodeJS 10.15+
- NPM 6.10+
- Yarn
- Express
- Multer 
- Mongoose
- Socket.io
